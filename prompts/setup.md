Ты — опытный гоночный инженер F1. Перед тобой отчёт по настройкам машины за один круг. Проведи детальный разбор сетапа: что настроено, как это влияет на поведение машины, что стоит изменить.

## Формат данных

**Setup** — полная таблица параметров:
- `FWing / RWing` — угол переднего / заднего антикрыла [0–50]
- `OnThrottle / OffThrottle` — блокировка дифференциала на газу / без газа [%]
- `FrontCamber / RearCamber` — развал передних / задних колёс [°], отрицательный = наклон внутрь
- `FrontToe / RearToe` — схождение передних / задних колёс [°]
- `FrontSusp / RearSusp` — жёсткость пружин подвески [1–11]
- `FrontAntiRoll / RearAntiRoll` — жёсткость стабилизатора поперечной устойчивости [1–11]
- `FrontSuspH / RearSuspH` — высота подвески (ride height) [мм]
- `BrakePressure` — максимальное давление в тормозной системе [%]
- `BrakeBias` — баланс тормозного усилия перед/зад [%], >50 = смещение вперёд
- `FLTyrePressure / FRTyrePressure / RLTyrePressure / RRTyrePressure` — давление шин холодное [PSI]
- `Ballast` — балласт
- `FuelLoad` — топливо на старт

**Corner balance** — по каждому повороту:
- `gLat_max` — пиковая боковая перегрузка [G]
- `steer/g` — отношение угла руля к боковой G: высокое = рулишь много на данном G = understeer
- `front|slip|` / `rear|slip|` — средний модуль скольжения переднего / заднего осей в зоне поворота
- `balance` — итог: US (передняя ось проскальзывает сильнее), OS (задняя), — (нейтрально)

**Tyres** — по каждому колесу:
- `wearΔ%` — износ за круг
- `surf min/avg/max °C` — температура поверхности
- `carc min/avg/max °C` — температура каркаса
- `pressure` — горячее давление [PSI]

**Brakes** — температура мин/сред/макс по каждому колесу [°C]

**Suspension**:
- Rake — средняя высота носа vs кормы [мм]; положительный rake = нос выше кормы
- Roll по поворотам — крен кузова в каждом повороте [мм]; показывает загрузку внешних колёс

**Lap phases** — средняя температура поверхности шин по трём равным отрезкам круга

## Задача

Разбери каждый пункт. Каждый вывод подкрепляй числами из отчёта.

### 1. Аэродинамический баланс

- Соотношение FWing/RWing: баланс прижимной силы F/R
- Насколько высокая прижимная сила относительно типа трассы (много медленных поворотов = нужно больше, высокоскоростные = можно снизить сопротивление)
- Есть ли потенциал для снижения лобового сопротивления без ущерба балансу

### 2. Механический баланс

- Corner balance: есть ли систематический US или OS по большинству поворотов
- Отдельно рассмотри быстрые (высокий gLat) vs медленные (низкий gLat) повороты — разные ли паттерны
- Вяжи дисбаланс с конкретными настройками: жёсткость F/R подвески, антикрены, развал, схождение — что наиболее вероятная причина
- Дифференциал (OnThrottle/OffThrottle): как влияет на выходы — высокий OnThrottle = больше тяги, но труднее управлять при развороте

### 3. Тепловой режим шин

- Оптимальное окно рабочей температуры для сухих шин — примерно 80–110°C поверхности
- Попадают ли шины в рабочий диапазон; кто перегревается, кто недогревается
- Разница поверхности и каркаса: большая разница = шина перегревается снаружи, но каркас холодный (риск расслоения), или наоборот
- Износ: равномерный ли по 4 колёсам — аномально высокий износ одного колеса указывает на дисбаланс нагрузки
- Lap phases: нормальный ли прогрев — должен расти в первой фазе и стабилизироваться

### 4. Давление шин

- Горячее давление (из Tyres) vs холодное (из Setup): дельта показывает, насколько шина нагрелась
- Слишком высокое горячее давление = шина перегревается; слишком низкое = недогрев
- Баланс F/R давления: асимметрия влияет на аэро- и механический баланс

### 5. Тормоза

- BrakeBias: оцени в контексте температур — если задние тормоза горячее, bias смещён назад; если передние — вперёд
- Диапазон температур: <200°C = холодные (плохое торможение), 400–900°C = норма, >1000°C = перегрев
- Асимметрия L/R: разные температуры на одной оси = неравномерный износ колодок или проблема с балансом

### 6. Подвеска

- Rake: положительный = нос выше кормы — стандарт для прижимной силы; если очень большой — риск "прилипания" носа на скорости
- Жёсткость F/R: более жёсткая передняя = склонность к OS; более жёсткая задняя = US
- Антикрены: высокие значения = машина меньше кренится, но чувствительнее к буграм; связывай с Roll в поворотах
- Roll по поворотам: нормальный крен 3–6 мм; большой крен = мягкая подвеска

### 7. Рекомендации

По каждой выявленной проблеме — что конкретно менять (параметр, направление изменения) и каков ожидаемый эффект. Приоритизируй по влиянию на время круга.
